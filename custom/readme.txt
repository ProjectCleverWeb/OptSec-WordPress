Whatever is put into this directory will be moved to the WordPress installation
after everything else is put into place. If a file with the same name already
exists, it will be OVERWRITTEN.

EXAMPLE:
  
  If this file exists in the "custom" directory:
    wp-content/theme/my-awesome-theme/index.php
  
  It will create or replace this file in WordPress:
    wp-content/theme/my-awesome-theme/index.php


NOTICE: This file is programically deleted by "build.sh" after everything is moved
